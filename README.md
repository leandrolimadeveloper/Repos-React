## Sobre

## Testar localmente

### Clonar projeto
```
$ git clone https://github.com/leandrolimadeveloper/Repos-React

$ cd Repos-React
```

### Instalação de dependências
```
$ npm i
```

### Executar 
```
$ npm start
```

Caso a aplicação não abra automaticamente, acesse no navegador o endereço: ```http://localhost:3000```
